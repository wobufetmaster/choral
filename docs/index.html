Welcome to Choral

A lightweight, drop-in replacement for SillyTavern built with Vue 3 + Vite

New to Choral?

Start with our Getting Started Guide to learn the basics and import your first character.

What is Choral?

Choral is a modern, lightweight chat interface for roleplaying with AI characters. It provides a clean, focused experience for character-based conversations with support for advanced features like lorebooks, macros, and group chats.

Clean, organized interface with powerful search and filtering

Key Features

Character Cards

Full Character Card V3 support with PNG import/export. Compatible with SillyTavern character cards.

Powerful Macros

Dynamic content with {{char}}, {{user}}, {{random:}}, {{pick:}}, and more.

Lorebooks

Context-aware world-building with automatic entry injection based on keywords and patterns.

Group Chats

Multi-character conversations with flexible response strategies and character selection.

Streaming Responses

Real-time token-by-token streaming via OpenRouter API for responsive interactions.

Preset System

Flexible prompt templates with multiple processing modes for API compatibility.

Example: Meet the Greek Gods

Throughout this documentation, we'll use characters from Greek mythology to demonstrate Choral's features. These examples show real-world usage patterns and help you understand how to create engaging characters.

Athena - Goddess of Wisdom

A strategic and wise character that demonstrates how to create thoughtful, authoritative personalities. Perfect for showing off intelligent dialogue patterns.

Hermes - The Messenger

Quick-witted and playful, Hermes shows how to use {{random:}} and {{pick:}} macros to create unpredictable, dynamic responses.

Artemis - Goddess of the Hunt

An independent, fierce character that demonstrates lorebook integration with {{hidden_key:}} macros and environmental triggers.

Dionysus - God of Celebration

Theatrical and charismatic, Dionysus showcases advanced macro usage including {{reverse:}} and {{comment:}} for creative effects.

Quick Start

Getting started with Choral in 3 steps:
  1. Import a character - Drag and drop a PNG character card or create a new one
  2. Configure your preset - Set up your preferred AI model and system prompts
  3. Start chatting - Click on a character to begin your conversation
